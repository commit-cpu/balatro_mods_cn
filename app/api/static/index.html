Balatro Mod Localization

Balatro CN

一个面向 Balatro 模组的中文汉化索引、自动补全和人工 review 工作台。

What is Balatro CN?

Balatro CN 用来追踪模组仓库、分析本地化状态，并把 AI 生成后仍需要人工判断的翻译集中到一个工作台。

Key Features

Mod Index

从 balatro-mod-index 读取名称、仓库、星标、分类和依赖信息。

Localization Status

对比 en/default 与 zh_CN，显示无汉化、部分汉化和完全汉化。

AI Pipeline

只补缺失 keys，减少重复 API 调用，并把未 review 的结果留给人工确认。

GitHub Workflow

后续可 fork 仓库、提交 zh_CN.lua、发起 PR，并跟踪上游 merge 状态。

Project

这个页面不是模组管理器，而是围绕“中文汉化覆盖率”和“翻译 review”建立的索引与流水线前端。

FAQ

状态数据来自本地探测报告和数据库。未探测的仓库不会假装已经完成，会保守显示为跳过或无汉化。

Acknowledgements

视觉方向参考 Balatro 的像素质感；工作流基于本仓库里的翻译、探测、GitHub 和 review 脚本。
